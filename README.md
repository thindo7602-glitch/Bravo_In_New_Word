## 📚 About

This repository is dedicated to **Brave New World**, one of the most influential dystopian novels ever written.

It serves as a clean, organized digital archive for reading, studying, and exploring the novel.

---

## ✨ Features

- 📖 Chapter navigation
- 🔍 Search support
- 🌙 Dark mode
- 📱 Responsive design
- 📑 Table of contents
- 🖼️ Illustrations (optional)
